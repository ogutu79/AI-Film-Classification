AI Pre-Screening & Classification System

The KFCB AI Pre-Screening System is a web-based application that automatically analyzes video content and provides content safety insights and age classification recommendations.

The system uses computer vision and lightweight AI techniques to detect potentially unsafe content such as:

Nudity (skin detection)

Violence (red/blood indicators)

General unsafe visual patterns

It generates:

📊 Visual reports (charts & dashboards)

🚩 Flagged scenes (with thumbnails)

🎯 Predicted age rating (G, PG, 16+, 18+

Objectives

Automate film pre-screening processes

Reduce manual review workload

Provide data-driven classification support

Improve consistency in content rating decisions

⚙️ Features
🎥 Video Analysis

Frame-by-frame sampling (~1 frame per second)

Skin detection (nudity estimation)

Red color detection (violence indicator)
